# Site Institucional - Louise Nakase

Site institucional desenvolvido para divulgação de serviços de psicologia clínica e reabilitação.

## Tecnologias utilizadas

- React 19
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Framer Motion
- React Icons

## Funcionalidades

- Apresentação profissional
- Informações sobre atendimentos
- Seção de serviços
- Contato via WhatsApp
- Animações e transições suaves
- Layout responsivo para celular, tablet e desktop

## Deploy

Projeto hospedado na Vercel.
